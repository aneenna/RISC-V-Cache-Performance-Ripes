# RISC-V Cache Performance Analysis using Ripes

## Project Description
This project analyzes cache performance using the Ripes RISC-V simulator.

## Objectives
- Compare Direct Mapped, 2-Way, and Fully Associative cache
- Measure hit rate and miss rate
- Analyze CPI impact
- Study spatial and conflict locality

## Tools Used
- Ripes Simulator
- RISC-V Assembly

## Experiments
1. Spatial Locality Test
2. Conflict Miss Test
3. Loop Intensive Memory Access
4. Hazard Test

## Expected Results
- Fully associative cache gives best performance
- 2-way reduces conflict misses
- Direct mapped has more conflict misses
